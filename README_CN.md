# Tarot Oracle 塔罗神谕
一个基于 AI 的塔罗牌占卜网站。

用户可以随机抽取三张塔罗牌，输入自己的问题，并获得由 AI 生成的专属塔罗解读

# 项目介绍
这是我第一次尝试使用 AI Coding 工具进行完整产品开发👏

项目从构思、页面生成、功能实现，到 GitHub 与 Vercel 部署，均通过 Claude Code 辅助完成。

现代人总被生活压力、情绪焦虑和前路迷茫裹挟，也正因为这样，塔罗牌这个想法一下子触动了我。

希望这个安静的小站点，能给你一点慰藉与指引，也愿你喜欢这份小小的心意~

## 功能特色
* 随机抽取三张塔罗牌
* 支持正位 / 逆位
* 输入个人问题进行占卜
* AI 自动生成塔罗解读
* 神秘、极简的女性向 UI 风格
* 支持网页端访问
  
## 技术栈
* React
* Vite
* TailwindCSS
* Claude API / Gemini API
* GitHub
* Vercel
