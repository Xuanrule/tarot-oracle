# Tarot Oracle 🔮
An AI-powered tarot divination website.

Users can randomly draw three tarot cards, enter their own questions, and receive personalized AI-generated tarot interpretations.

## Project Intro
This is my first attempt to build a complete product with AI coding tools.👏

From concept design, page layout and feature implementation to deployment on GitHub and Vercel, the whole project was developed with the assistance of Claude Code.

Modern people are often trapped by life pressure, emotional anxiety and confusion about the future. 

That’s why the idea of creating a tarot website deeply resonated with me.

Hope you enjoy it :)

## Features
* Randomly draw three tarot cards
* Support upright / reversed card positions
* Divination by entering your personal question
* Automatic tarot interpretation generated by AI
* Mysterious, minimalist and feminine UI style
* Fully accessible on web browsers

## Tech Stack
* React
* Vite
* TailwindCSS
* Claude API / Gemini API
* GitHub
* Vercel
